Data: 26/03/2026

## Introdução
Este relatório apresenta o processo de implementação de ferramentas na empresa abstergo, realizado por Cesar Augusto Pereira da Silva. O objetivo do projeto foi elencar 3 serviços AWS, com a finalidade de realizar diminuição de custos imediatos.

## Descrição do Projeto
O projeto de implementação de ferramentas foi dividido em 3 etapas, cada uma com seus objetivos especí­ficos. A seguir, serÃ£o descritas as etapas do projeto:

Etapa 1: 
- Ferramenta: Amazon S3 One Zone-IA
- Foco: Economia de custos no armazenamento
- Descrição: Por se tratar de um upgrade no armazenamento de dados de servidor físico para servidor em nuvem, recomendo a ferramenta acima por um período de adaptação dos usuários, podendo ser definitivo se seu desempenho se demonstrar superior às necessidades da empresa no momento.

Etapa 2: 

- Ferramenta: Amazon S3 Standard
- Foco: Armazenamento de dados
- Descrição: Após o período de adaptação e conforme a necessidade, faremos o upgrade de ferramenta tendo em vista que a busca por dados se tornará mais frequente conforme a expansão da área atendida.

Etapa 3: 
- Ferramenta: Amazon Relational Database Service
- Foco: Gerenciamento de banco de dados
- Descrição: É uma ferramenta voltada para a facilitação da compreensão das informações tendo como linguagem base de consulta o SQL



## Conclusão
A implementação de ferramentas na empresa Abstergo Industries tem como esperado a redução de custos com manutenção e armazenamento de servidores físicos, aumentar a segurança dos dados e facilitar a compreenaão dos dados gerados para ajudar na tomada de decisões dos gestores, o que aumentará a eficiência e a produtividade da empresa. Recomenda-se a continuidade da utilização das ferramentas implementadas e a busca por novas tecnologias que possam melhorar ainda mais os processos da empresa.
